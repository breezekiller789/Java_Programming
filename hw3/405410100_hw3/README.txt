1 . 編譯方法
    
    我有放Makefile，助教只要打入make，就可以產生class file

    $java HW3   即可執行程式

2 . 程式說明

    詳細的程式運作流程我都寫在程式裡面，程式跑起來的介面，左邊是文章，看使用者

    想要練習哪篇文章，上面就是該篇文章的文字，右邊的“錯誤”代表目前使用者打錯了

    幾個字，中間空白的地方就是使用者可以打字的地方，只要字元打對的話就會自動跳

    下一個字元繼續打入，打錯就會在左上角跳出提示說打錯了，然後右上角的錯誤會加

    一，如果所有文章都打完的話，我會顯示使用者打字的準確率是幾%。

3 . Trace Code

    我有附上tags檔案，方便助教來追蹤程式碼，如果助教使用Vim來看code的話，可以在

    Vim裡面打入 :tag <function_name> 來追蹤函式的定義，打入這個會直接跳到函式定

    義的地方，如果要跳回來的話，<C-o>就可以跳回來。然後，在打入函式名稱的時候，

    可以用tab來autocomplete，滿方便的。
